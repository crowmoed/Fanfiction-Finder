The search composer, eight ways

Eight structural skeletons for the FicFinder search bar, same four capabilities each time: query, fandom, strict filters, submit. No history tray in any of them; history moves to the sidebar. Every demo is live: type in it, focus it, submit it to run the real pipeline stages.

Skeleton register on purpose. Pick the structure, not the paint. All motion honors reduced-motion.
